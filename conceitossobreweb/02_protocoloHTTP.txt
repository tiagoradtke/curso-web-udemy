> O Hypertext Transfer Protocol, sigla HTTP
    - Camada de Aplicação
    - Stateless (não tem estado, toda conexão é como se fosse a primeira vez)
    - Cliente-Servidor
    - TCP/IP

    - Fluxo Básico
        1) Usuário Informa URL
        2) Browser Gera a Requisição
        3) Servidor Web gera a resposta
        4) Browser renderiza a página
    
    - Métodos do HTTP
        - get
        - post
        - put
        - delete
        - trace
        - options
        - connect
        - head
    - Grupos de Status HTTP
        - 1xx | Informativa
        - 2xx | Sucesso
    	- 3xx | Redirecionamento
        - 4xx | Erro de cliente
       	- 5xx | outros erros

    - Sempre bom diminuir quantidade de requisições, 
    minificação de arquivos indicado
    https://pt.wikipedia.org/wiki/Handshake
